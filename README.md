# versekin-shots

Public screenshot assets for **iconbpv/versekin** (a private repo).

GitHub only renders images inline in issue/PR markdown from its own user-attachments
CDN or from a **public** raw URL. Since versekin is private, its own raw URLs are
blocked by Camo. Screenshots for versekin PRs live here instead.

**Convention:** put PNGs under `<issue-or-pr-number>/<name>.png`, then embed in the
versekin PR body:

    ![caption](https://raw.githubusercontent.com/iconbpv/versekin-shots/main/<issue>/<name>.png)

Only app-UI screenshots — nothing sensitive (this repo is public).
